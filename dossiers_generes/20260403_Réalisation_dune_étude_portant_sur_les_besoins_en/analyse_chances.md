## 3. FORCES / FAIBLESSES / RISQUES / RECOMMANDATIONS

### ✅ FORCES
- **Légitimité tech réelle** : Expertise IA + livre ENI + Hub France IA
- **Références institutionnelles solides** : CDC, Action Logement, CCI = crédibilité OPCO
- **Approche innovante** : Utilisation IA pour analyse données + méthodologie structurée
- **Agilité** : Structure souple, réactivité vs gros cabinets
- **Certifications** : Qualiopi + RS6776 = gage qualité formation
- **Couverture géographique** : Paris + Annecy + réseau national

### ❌ FAIBLESSES
- **Pas d'expertise directe No/Low-code** : Risque de manque de crédibilité technique
- **Nouveau sur marchés publics** : Manque d'expérience procédures administratives
- **Portfolio études limité** : Principalement formation, pas d'études sectorielles référencées
- **Équipe encore réduite** : Peut questionner la capacité à mener une étude d'envergure nationale

### ⚠️ RISQUES
- **Sous-estimer la complexité** : Une étude nationale No/Low-code nécessite ressources importantes
- **Concurrence experte** : Cabinets spécialisés études RH/compétences très rodés
- **Critères notation inconnus** : Difficile d'optimiser sans connaître les pondérations
- **Budget non communiqué** : Risque de sur/sous-évaluation prix

### 🎯 5 RECOMMANDATIONS CONCRÈTES

1. **RECRUTER UN EXPERT NO/LOW-CODE** en freelance pour co-signer l'étude (renforcer crédibilité technique)

2. **METTRE EN AVANT L'INNOVATION IA** : Proposer analyse automatisée des offres d'emploi + scraping formations + IA pour identifier tendances

3. **VALORISER LES RÉFÉRENCES INSTITUTIONNELLES** : Insister sur CDC, Action Logement = connaissance enjeux publics

4. **PROPOSER MÉTHODOLOGIE HYBRIDE** : Étude quantitative (IA) + qualitative (interviews terrain via notre réseau)

5. **S'ASSOCIER À UN PARTENAIRE ÉTUDES** : Envisager consortium avec cabinet spécialisé RH pour combiner expertises
